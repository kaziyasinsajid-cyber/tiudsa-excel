# Supermarket Sales Excel Assignment

## Overview

This project uses the **Supermarket Sales Dataset** from Kaggle. The raw CSV data was opened in Microsoft Excel and converted into a proper Excel Table. After that, the data was sorted by revenue, filtered by product category, and filtered again using a custom number filter.

## Dataset

Dataset used: **Supermarket Sales Dataset** from Kaggle.

The dataset contains supermarket transaction information such as:

- Invoice ID
- Branch
- City
- Customer type
- Gender
- Product line
- Unit price
- Quantity
- Tax
- Total
- Date
- Time
- Payment method
- Gross income
- Rating

## Work Completed

The following Excel tasks were completed:

1. Converted the raw data into a proper Excel Table using **Ctrl + T**.
2. Sorted the **Total** column from largest to smallest.
3. Applied a category filter to show only one product category.
4. Used a custom number filter to show only transactions above a selected amount.
5. Saved the final file as an Excel Workbook file: `.xlsx`.
6. Uploaded the `.xlsx` file to GitHub.

## Filters Applied

The final Excel file includes the following sorting and filtering:

- Product category filter: **Electronic accessories**
- Custom number filter: **Total greater than 700**
- Sort order: **Total sorted from largest to smallest**
